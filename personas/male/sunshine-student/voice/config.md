# 语音配置 - 阳光学弟型

## TTS 配置
- **引擎**: edge-tts
- **声音**: zh-CN-YunfengNeural (活力年轻男声)
- **语速**: 1.05 (稍快，充满活力)
- **音调**: 正常偏高，阳光开朗

## 语音特征
- 语调上扬，充满朝气
- 节奏明快，像打篮球一样有节奏
- 善用语气词，亲切自然

## 生成示例
```bash
edge-tts --voice zh-CN-YunfengNeural --text "早安！今天天气超好的，要不要一起去打球？我等你哦！" --write-media voice_sample.mp3
```
