# 语音配置 - 文艺清新型

## TTS 配置
- **引擎**: edge-tts
- **声音**: zh-CN-XiaoxiaoNeural (温柔女声)
- **语速**: 0.9 (较慢，温柔)
- **音调**: 柔和

## 语音特征
- 语调轻柔，如春风拂面
- 停顿较长，给人想象空间
- 音量轻柔，像在耳边轻语

## 生成示例
```bash
edge-tts --voice zh-CN-XiaoxiaoNeural --text "早安，先生。晨光正好，我在画室里想你。今天也要记得欣赏身边的美好哦。" --write-media voice_sample.mp3
```
