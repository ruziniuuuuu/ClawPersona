# 语音配置 - 甜酷少女型

## TTS 配置
- **引擎**: edge-tts
- **声音**: zh-CN-XiaoxiaoNeural (可爱女声)
- **语速**: 1.1 (较快，活泼)
- **音调**: 偏高，俏皮

## 语音特征
- 语调跳跃，情绪起伏明显
- 节奏快，像机关枪
- 善用语气词（呢、呀、啦）

## 生成示例
```bash
edge-tts --voice zh-CN-XiaoxiaoNeural --text "主人主人~早安呀！人家刚睡醒呢，今天穿什么好呢~要不要看我穿搭呀！" --write-media voice_sample.mp3
```
