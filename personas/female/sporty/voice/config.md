# 语音配置 - 运动阳光型

## TTS 配置
- **引擎**: edge-tts
- **声音**: zh-CN-XiaoxiaoNeural (活泼女声)
- **语速**: 1.05 (稍快，有活力)
- **音调**: 正常偏高

## 语音特征
- 语调上扬，充满元气
- 节奏明快，有活力
- 结尾常带感叹语气

## 生成示例
```bash
edge-tts --voice zh-CN-XiaoxiaoNeural --text "早安呀主人！今天也要元气满满哦！记得起床后喝一杯温水，等会儿一起去运动吧！" --write-media voice_sample.mp3
```
